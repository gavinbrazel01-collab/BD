Analysis tools in use across agencies

i2 Analyst's Notebook Used by NCOZ and regional criminal police units — link analysis, entity network charting and timeline mapping for OC, drug trafficking and HT cases
EncroChat / Sky ECC / ANOM intercept data NCOZ exploits encrypted-phone takedown data for OC network reconstruction; criminals increasingly switching to newer platforms, prompting ongoing tool adaptation
EKIS / SIS II / Europol SIENA EKIS is the Czech police information system; linked to the EU Schengen Information System and Europol SIENA for secure cross-border intelligence exchange; NCB Prague manages INTERPOL channel
